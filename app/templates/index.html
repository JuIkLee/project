How it works

VisionAI provides services that can diagnose human eye fundus diseases based on AI deep learning technology. It analyzes fundus photos among medical data that can diagnose ophthalmic diseases. VisionAI uses a model that divides ocular fundus photographs data containing 8,000 sheets provided by Kaggle into four categories (normal, cataract, glaucoma, and retinal disease). The learning model applies CNN (a convolutional neural network: a model specialized for image processing) among deep learning algorithms, and the network consists of an Efficient net. After that, as an optimization process, a web page is built by applying the developed model while tuning hyperparameters. The diagnosis results of the fundus photos are provided through the web page.
